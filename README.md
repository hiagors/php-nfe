# php-docker

## :rocket: Começando
``` bash
  # Clonar o projeto:
  $ git clone https://github.com/hiagors/php-nfe.git

```

## :gear: Executando docker-compose
```bash
  # Entrar no diretório do repositório:
  $ cd php

  # Executar o docker compose:
  $ docker-compose up

  # Rodar o container docker:'
  $ docker run php

```
